# RevPlay Project 

RevPlay is a console-based music streaming application developed using Java.  
It allows users to explore songs, manage playlists, albums, podcasts, and track listening history.

## 🛠 Technologies Used
- Java
- JDBC
- Oracle SQL
- Log4j2
- JUnit
- Maven

## 📌 Features
- User registration and login
- Song search and playback
- Playlist management
- Album and podcast management
- Listening history tracking
- Logging using Log4j2
- Unit testing using JUnit

## 🏗 Project Architecture
- DAO Layer – Handles database operations
- Service Layer – Business logic
- Model Layer – Entity classes
- UI Layer – Console-based interaction

## ▶ How to Run
1. Import the project into Eclipse / IntelliJ
2. Configure Oracle database and JDBC connection
3. Run the `MainMenu` class

## 🧪 Testing
- Unit tests written using JUnit
- Test cases for DAO and service layers
